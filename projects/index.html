Projects

Computer club members are constantly involved with hardware and software related projects.

CCoWMU.org

The computer club website is an ongoing project maintained entirely by members. The site is developed at the CCoWMU GitHub organization allowing for easy collaboration.

PLAN

The computer club hosts a twice-annual LAN party at the Parkview campus. Members are responsible for putting together the PLAN website and the event itself.

IRC Bots

The computer club has an active IRC server, and members are encouraged to develop their own IRC bots. Some boilerplate is available to get members started.
